# Khoshghadam Frontend

## Setup

Make sure to install the dependencies:

```bash
bun i
```

## Development Server

Start the development server on `http://localhost:8080`:

```bash
bun serve
```

## Production

Build the application for production in `SSR` mode:

```bash
bun run build
```

Build the application in `SPA` mode.

```bash
bun run generate
```
